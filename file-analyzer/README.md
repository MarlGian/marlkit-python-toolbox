# File Analyzer CLI Tool

A simple Python command-line utility to analyze text files.  
Counts letters, words, numbers, and shows word frequency. Perfect for learning Python, practicing file handling, and building CLI tools.

---

## Features

- Count total letters in a file  
- Count total words in a file  
- Count total numbers in a file  
- Show frequency of each word in a file (alphabetically sorted)  
- Works directly from the command line  

---

## Requirements

- Python 3.x  
- Works on Windows, macOS, Linux  

---

## Usage

```bash
python file-analyzer.py <filename> <flag>
